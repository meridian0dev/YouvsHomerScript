# You vs Homer Script

Script for the Roblox game "You vs Homer" with nice features and full anticheat bypass.

## Features

### Automation
- **Coin Farm** - Automatically farm coins while AFK

### Customization
- **Skin Changer** - Buy and equip any skin in the game
- **FOV Changer** - Adjustable field of view with slider

### Visuals
- **ESP System**
  - Bart ESP
  - Homer ESP
  - AFK Player ESP
  - Dead Player ESP
  - Name ESP with multiple display modes
  - Distance indicators
  - Customizable colors
- **Fullbright** - See clearly in any lighting condition
- **No Fog** - Remove environmental fog

### Player
- **Walkspeed** - Adjustable movement speed
- **Jump Power** - Enhanced jumping
- **Infinite Jump** - Jump infinitely with hold delay
- **Gravity** - Custom gravity settings
- **Fly** - Full flight control (PC only)

### Security
- **Full Anticheat Bypass** - Bypasses Adonis and local anticheats
- **Anti-kick protection**

### Utilities
- **Rejoin** - Quick rejoin current server
- **Teleports** - Instant teleportation to key locations

## Requirements

- A script executor (Velocity, Bunni, DeltaExploits)
- Roblox game: You vs Homer

## Configuration

All features can be toggled and customized through the in-game GUI.
